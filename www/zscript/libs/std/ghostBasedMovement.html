int ConstWalk4(npc n, int counter, bool flying = false) 🔗 Source

Pass a variable for the ‘counter’ param which is initialized to -1, and assign the return from the function back to the counter variable. Ex:

int counter = -1;
counter = ConstWalk4(n, counter);
