Function descriptions#

void __Tango_SetCurrentSlot(int slot)

Loads some commonly used data about the current slot into __Tango_Data. Kind of ugly, but it beats recalculating the numbers whenever theyre needed or passing them into every function.

bool __Tango_IsSpecialDataMarker(int character)

Returns true if this character is a special data marker. End markers dont count.

void __Tango_IncScreenFreeze()

Increment the screen freeze counter. Freeze the screen if it was 0.

void __Tango_DecScreenFreeze()

Decrement the screen freeze counter. Unfreeze the screen if its now 0.

int __Tango_FloatToASCII(char32[] buffer, int value, bool allDigits)

string.zhs float-to-text functions dont work right in 2.50, so this is used instead.
