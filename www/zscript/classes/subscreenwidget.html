int RequiredCounter 🔗 Source

value CounterIndex (CR_)

The counter this widget requires to be active/visible. Must relate to the RequiredCounterValue based on the RequiredCounterCond.

  • If CR_NONE, no requirement will be checked.

  • If SUBW_GENFL_REQ_COUNTER_PERC is set, then the percentage this counter is full will be compared

  • Otherwise, if SUBW_GENFL_REQ_COUNTER_MAX is set, then the max value of this counter will be compared

  • Otherwise, this counter’s value will be compared

int RequiredCounterCond 🔗 Source

value ConditionType (CONDTY_)

The counter operation this widget requires to be active/visible. The RequiredCounter must relate to the RequiredCounterValue by this operation. If CONDTY_NONE, no requirement will be checked.

int RequiredCounterValue 🔗 Source

The counter value this widget requires to be active/visible. The RequiredCounter must relate to this value based on the RequiredCounterCond.

int RequiredLItems 🔗 Source

value LevelItemBitflags (LI_)

The Level Items that are required to be owned (or unowned, if SUBW_GENFL_REQ_INVERT_LITEM is set) If SUBW_GENFL_REQ_ANY_LITEM is set, any state is required instead of every state.

int RequiredLItemLevel 🔗 Source

The Level that RequiredLItems are checked for. If set to ‘-1’, uses the current level.

bool[] RequiredLStates 🔗 Source

index 0-31

The Level States that are required to be set (or unset, if SUBW_GENFL_REQ_INVERT_LSTATE is set) If SUBW_GENFL_REQ_ANY_LSTATE is set, any state is required instead of every state.

int RequiredLStateLevel 🔗 Source

The Level that RequiredLStates[] are checked for. If set to ‘-1’, uses the current level.

int RequiredScreenStateMap 🔗 Source

The Map to require screen states or exstates for.

int RequiredScreenStateScreen 🔗 Source

The Map to require screen states or exstates for. If ‘-1’, uses the current map+screen, ignoring RequiredScreenStateMap.

index ScreenState (ST_)

The Screen States that are required to be set (or unset, if SUBW_GENFL_REQ_INVERT_SCRSTATE is set) If SUBW_GENFL_REQ_ANY_SCRSTATE is set, any state is required instead of every state.

bool[] RequiredScreenExStates 🔗 Source

index 0-31

The Screen ExStates that are required to be set (or unset, if SUBW_GENFL_REQ_INVERT_SCRSTATE is set) If SUBW_GENFL_REQ_ANY_SCRSTATE is set, any state is required instead of every state.
