Variable descriptions#

int Max 🔗

Max number of itemsprites currently allowed.

int ID 🔗

The item ID to give when the item is collected.

int Type 🔗

The type itemdata of the item. Uses the IC_ constants.

int Family 🔗

The type itemdata of the item. Uses the IC_ constants.

int Level 🔗

The item’s level

int OriginalTile 🔗

The starting tile of the itemsprite’s animation.

int FlashCSet 🔗

The secondary CSet used for flashing.

int NumFrames 🔗

The number of frames in the itemsprite’s animation.

int Frame 🔗

The current frame number that the animation is on.

int ASpeed 🔗

The speed of the animation, in frames per frame.

int Delay 🔗

The number of delay frames at the start of the itemsprite’s animation.

bool Flash 🔗

If the itemsprite is flashing or not. When flashing, it changes between its’ CSet and FlashCSet.

int Pickup 🔗

The item pickup flags, as a bitwise flagset. Use the IP_ constants to access. Take care when using any INTERNAL flags.

bool NoSound 🔗

If true, picking up the item does not play it’s usual pickup sound assigned in the item editor. Does not affect holdup sound.

bool NoHoldSound 🔗

If true, picking up the item does not play the screen’s holdup sound. Does not affect the usual pickup sound assigned in the item editor.

untyped[] InitD 🔗

The 8 InitD[] arguments for the itemsprite script.

int AClock 🔗

The timer that ASpeed is used with.

int PickupString 🔗

The message string to display upon picking up the item.

int PickupStringFlags 🔗

The flags associated with the pickup string, as bitwise flags. Use the IPSTR_ constants to access this.

int Script 🔗

The itemsprite script running on this itemsprite.

int DroppedBy 🔗

The dropset that this item was dropped by. Will be set for engine-dropped items, can be written by scripts as well.

bool ForceGrab 🔗

If set to true, the item will automatically be collected by the player as soon as possible.
