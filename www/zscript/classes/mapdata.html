Variable descriptions#

int Valid

Returns true if the screen is valid. Screens that appear with the default blue background in ZQuest are invalid. Modifying a combo on a screen makes it become valid. If a layer is invalid, it will not be drawn.

int Guy

The screen guy.

int String

The screen string.

int RoomType

The screen room type. Use the RT_ constants for this value.

int Item

The item placed on the screen. -1 if no item placed.

int TileWarpType

The Tile Warp type for Tile Warps A, B, C, and D; 0, 1, 2, and 3 respectively.

Valid values: WarpType (WT_).

int Script

The screen script that runs on this screen.

untyped InitD

The 8 script arguments for the script that runs on this screen.

bool[] TileWarpOverlay

Set or get the overlay state for the warp ID corresponding to the index of this array.

int DoorComboSet

The Door Set used for the NES dungeon doors on this screen.

int WarpReturnX

The X-component for each of the four 2.50+ (blue) warp return squares.

int WarpReturnY

The Y-component for each of the four 2.50+ (blue) warp return squares.

int StairsX

The X/Y position of the Stairs secret on the screen.

int StairsY

The X/Y position of the Stairs secret on the screen.

int ItemX

The X/Y position the screens item spawns at.

int ItemY

The X/Y position the screens item spawns at.

int Palette

The palette set in the F4 menu in ZQuest for this screen. Has no effect during play, but can be read and written.

int Door

The 4 door states of the screen.

Valid indices: DIR_UP, DIR_DOWN, DIR_LEFT, DIR_RIGHT

int TileWarpDMap

The destination DMap for each of the four warp types.

int TileWarpScreen

The destination screen for each of the four warp types.

int ExitDir

The Maze Path Exit Direction.

int Enemy

The 10 enemies that appear on this screen.

int Pattern

The spawn pattern for the screen enemies.

Valid values: EnemyPattern (PATTERN_).

int SideWarpType

The Sidewarp type for Sidewarps A, B, C, and D; 0, 1, 2, and 3 respectively.

Valid values: WarpType (WT_).

int SideWarpOverlay

Set or get the overlay state for the warp ID corresponding to the index of this array.

int WarpArrivalX

The X-component for the pre-2.50 (green) arrival square.

int WarpArrivalY

The X-component for the pre-2.50 (green) arrival square.

int MazePath

The four directions you must go for the maze path.

Valid indices: DIR_UP, DIR_DOWN, DIR_LEFT, DIR_RIGHT

int SideWarpScreen

The destination screen for each of the four sidewarps.

int SideWarpID

Returns the sidewarp ID for a given screen edge, as SideWarpIDdir;

Returns SIDEWARP_NONE if there is no sidewarp in that direction.

Valid indices: DIR_UP, DIR_DOWN, DIR_LEFT, DIR_RIGHT Valid values: SideWarpIndex (SIDEWARP_)

int TileWarpReturnSquare

The return square (A->D) used by Tile WarpA->D.

int SideWarpReturnSquare

The return square (A->D) used by Side WarpA->D.

int SideWarpDMap

The destination DMap for each of the four sidewarps.

int UnderCombo

The screens undercombo, which will appear as a result of various combo interactions, such as pushing blocks, awakening armos, etc.

int UnderCSet

The CSet associated with the undercombo.

int Catchall

The screens catchall value. This is the roomtype-specific data, such as the Special Item in a Special Item room.

int CSensitive

int NoReset

int LayerMap

untyped GetFFCInitD(int ffc_index, int n)

Returns the value of InitDn for the ffc on the screen with an ID of ffc_index.

This needs to be converted to the type untyped to comply with the change of float ffc->InitD] to untyped ffc->InitD[

Note: Expressed as a function due to lack of 2D arrays in ZScript.

With 2D arrays, this would simply be FFCInitD328.
